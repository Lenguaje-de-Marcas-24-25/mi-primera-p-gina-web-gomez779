¿Qué es HTML?

HTML, sigla en inglés de HyperText Markup Language (lenguaje de marcas de hipertexto), es el lenguaje de marcado para la elaboración de páginas web. Define la estructura básica y un código (denominado código HTML) para la definición de contenido en una página web, como texto, imágenes, videos, juegos, entre otros.
